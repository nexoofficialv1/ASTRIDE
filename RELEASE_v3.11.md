# ASTRIDE v3.11 — Offer & Campaign Module

## Added
- Admin Web Panel Offer & Campaign manager
- Driver, Passenger, Promoter and Area Promoter targeting
- Draft / Scheduled / Active / Paused / Expired lifecycle
- Area, city and ride-type targeting
- Promo code validation and passenger fare discount application
- Fixed bonus, flat discount, percentage discount, cashback, zero commission, target and referral reward types
- Ride-count and generic metric target engine
- Per-user limits, maximum reward and maximum payout budget
- Idempotent campaign redemption ledger
- Automatic ride-completion evaluation for Passenger, Driver and linked Partners
- Protected campaign-event API for onboarding, referral and area-growth achievements
- Persistent campaign state and SQL migration 011
- Admin audit events and campaign payout reporting
