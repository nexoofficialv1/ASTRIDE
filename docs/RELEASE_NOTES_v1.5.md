# v1.5 Native Location and Map Integration Foundation

- Android and iOS location permission manifests for passenger and driver apps.
- Driver background-location declarations and foreground-service permissions.
- Native GPS permission, current-position and navigation-grade stream service.
- Offline tracking queue and retry-capable ride tracker.
- Backend-controlled provider map widget with Google native map implementation and adapter boundary for Mappls/other providers.
- App version 1.5.0.

## Release gate
A signed build still requires Flutter/Android/iOS toolchains, valid map credentials, real devices, and platform-store permission review.
