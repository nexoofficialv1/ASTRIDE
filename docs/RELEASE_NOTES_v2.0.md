# ASTRIDE v2.0 — Mobile UI/UX Production Foundation

## Brand implementation
- Official name: ASTRIDE
- Company: Astra Technologies
- Tagline: Ride Smarter. Reach Faster.
- Colors: Navy #0D1B3D, Green #22C55E, Orange #FF8A00
- Typography contract: Poppins
- Material 3 component system

## Passenger UI foundation
- Branded map-first home screen
- Current-location and destination booking card
- Saved-place shortcuts
- Ride category selector (Toto, Auto, Bike)
- Fare/payment confirmation layout

## Driver UI foundation
- Branded online/offline dashboard
- Earnings and performance summary
- Full-screen incoming ride request
- Pickup/drop and fare presentation

## Release note
This milestone establishes the production design system and core screen implementation. Final APK/AAB/iOS builds still require Flutter SDK, platform signing, real map keys, Firebase files, and device testing.
