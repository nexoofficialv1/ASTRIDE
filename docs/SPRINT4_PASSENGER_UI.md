# ASTRIDE v3.1 Sprint 4 — Passenger UI Implementation

Implemented in Flutter source:

- Approved map-first Passenger Home dashboard
- Pickup and destination booking sheet
- Full Toto, Share Toto and Motorcycle selectors
- Cash, UPI and Both payment preference selector
- SafeRide toggle and summary
- Fare quote and booking hand-off
- Nearby-driver searching state
- Driver arriving / trusted driver card
- Live-trip route canvas and trip actions
- Call, chat, trip sharing and SOS controls
- English, Bengali and Hindi locale parity

## Validation boundary

The source and static UI contracts are validated. A real Flutter/Android compile and visual comparison on an emulator or physical phone remains required because Flutter SDK and Android SDK are not installed in this execution environment.
