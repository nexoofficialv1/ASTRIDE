# Feature Matrix

## Passenger
OTP/Google/Apple login; language selection; current location; autocomplete; local landmarks; pickup pin; destination; stops; fare estimate; book now/schedule; book for others; driver matching; live tracking; masked call/chat; OTP start; route deviation; SOS; trip share; cash/UPI/wallet; receipt; rating; history; complaints.

## Driver
OTP login; KYC; vehicle/documents; admin approval; online/offline; request timer; pickup navigation; arrived/waiting/no-show; OTP start; background GPS; ride completion; cash/online collection; earnings; commission; settlement; rating; support; document expiry.

## Admin
Live map; active rides; manual assignment; driver verification; passenger/driver control; fare and zone rules; coupons; payments; settlements; complaints; SOS desk; notifications; reports; role permissions; audit logs.
