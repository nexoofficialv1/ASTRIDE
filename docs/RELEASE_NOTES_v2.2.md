# ASTRIDE v2.2 — Driver App Complete UI/UX

## Included
- Driver language selection and OTP login
- Profile and vehicle registration
- Document upload contract and approval state
- Online/offline availability with approval gate
- Earnings-first driver dashboard
- Full-screen ride request with accept/decline
- Pickup arrival, passenger OTP, ride start and completion
- Ride history, wallet, earnings and settlement request
- Driver profile, vehicle, bank/UPI, support and safety sections
- English, Bengali and Hindi parity
- ASTRIDE navy/green/orange Material 3 design system

## Release boundary
This is the complete production UI/UX and API-contract implementation baseline. Real file picker/camera uploads, turn-by-turn native navigation, live provider credentials and signed Android/iOS binaries remain release-pipeline tasks.
