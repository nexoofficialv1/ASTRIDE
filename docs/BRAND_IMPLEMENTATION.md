# ASTRIDE Brand Implementation Contract

The supplied brand board is the visual reference. Mobile interfaces use the approved navy, ride green, energy orange, light background and Poppins typography. Product screens must not mix languages. All customer-visible labels continue to come from locale files. External provider identity must never replace ASTRIDE identity.

Provider logos may only appear inside payment/map legal attribution areas where required. Map, OTP, payment and notification providers remain backend-controlled.
