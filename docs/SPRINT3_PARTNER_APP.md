# ASTRIDE v3.0 Sprint 3 — Partner App

## Implemented
- One Flutter application for PROMOTER and AREA_PROMOTER roles.
- Secure partner login/session/logout API.
- Role-scoped dashboard and driver performance list.
- Date-range ready analytics API.
- Acceptance, rejection, cancellation and completed ride metrics.
- Coaching, encouragement, warning and training messages with audit records.
- Earnings summary and month-close withdrawal gate.
- Area Promoters inherit access only to drivers linked to their assigned area.

## Permission boundary
Partners cannot suspend drivers, edit fares, alter wallets, waive penalties or access global admin data.

## Build status
Flutter source is present and compile-ready in structure. Native APK compilation still requires Flutter/Android SDK or GitHub Actions.
